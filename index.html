Etsy Active Listing Browser

An example app using Angular JS, Google Material Design with angular-material. The app is 100% client side, and allows you to search and bookmark etsy active listings using the etsy JSONP REST API.

Currently best viewed in Google Chrome

Build & development

To get up and running:

  1. npm install
  2. npm install grunt-cli bower -g
  3. grunt serve
  4. Add your Etsy API Key in app/scripts/app.js

Run grunt for building, all files will be minified and gzip'd to /dist. Run grunt serve for running with livereload in development.

Generating Components

You can generate new controllers, models, routes, views, and tests with generator-angular.

  • Install the generators

npm install -g yo generator-karma generator-angular

  • Generate a controller called 'user'

yo angular:controller user

This will generator the boilerplate angular controller and tests. Full docs here for all available methods and generators available.

CSS Dependancies

All css dependancies are automatically wired from bower_components into index.html via grunt when launching or building. See here and here for details.

Local Storage

Data is stored via local storage driver, and uses $scope bindings provided by angular-local-storage. The bookmarks $scope map is bound to local storage, so there is no need to CRUD or interact with the local store, just update the $scope variable.

Testing

Running grunt test will run the unit tests with karma. All tests are located in the /test directory, to create new tests it's easiest if you use generator-angular, than modify as needed.
